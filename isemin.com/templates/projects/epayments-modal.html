Проект международной платежной системы - Epayments

Данный проект позволяет переводить платежи любому пользователю платежной системы. Клиент, получивший платеж, в свою очередь может снять деньги с карты этой платежной системы в любом банкомате мира.

Реализован данный проект был на ASP.NET MVC3 + Entity Framework 3. В качестве СУБД использовался SQL Server 2008. Тестовый проект был переписан на ASP.NET MVC 6 + NHibernate + Angular. В качестве СУБД - Postgres + Redis.
